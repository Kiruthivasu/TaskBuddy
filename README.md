# TaskBuddy 📝

TaskBuddy is a simple and responsive task management application built with React and Vite. It helps users organize daily tasks efficiently with features like task creation, editing, completion tracking, progress monitoring, and dark mode support.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete individual tasks
- 🔴 Clear all tasks with one click
- 📊 Task completion progress bar
- 🌙 Dark Mode / ☀️ Light Mode toggle
- 📱 Responsive user interface

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/taskman.git
```

2. Navigate to the project folder

```bash
cd taskman
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

## 🎯 How It Works

1. Enter a task in the input field.
2. Click **Add** to save the task.
3. Use **Edit** to modify a task.
4. Click **Complete** when a task is finished.
5. Delete unwanted tasks using the **Delete** button.
6. Track your productivity through the progress bar.
7. Switch between **Light Mode** and **Dark Mode** using the theme toggle button.
